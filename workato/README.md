# Workato Integration Platform

Workato recipes and integration workflows.

## Structure

- `recipes/` - Workato recipe definitions
  - `atomic-salesforce/` - Salesforce atomic recipes
  - `atomic-stripe/` - Stripe atomic recipes
  - `orchestrator-recipes/` - Orchestration workflows
  - `Salesforce/` - Salesforce integration recipes
- `scripts/` - Workato CLI and deployment scripts
  - `deploy/` - Deployment automation
  - `test/` - Testing utilities
  - `cli/` - CLI wrappers
- `docs/` - Workato integration documentation

## Usage

See individual recipe directories for specific integration documentation.
