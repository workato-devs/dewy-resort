# Shared CLI Tooling

Shared command-line tools and utilities for the monorepo.

## Structure

- `bin/` - CLI wrapper scripts
- `workato-cli/` - Workato CLI (gitignored)
- `sf-cli/` - Salesforce CLI (gitignored)
- `scripts/` - Shared utility scripts

## Installation

CLI tools are installed automatically when needed. See individual tool documentation for manual installation.
